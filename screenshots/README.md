# Screenshots

Place your application screenshots here with the following names:

- `dashboard.png` - Main dashboard screenshot
- `optimizer.png` - System optimizer screenshot  
- `startup.png` - Startup manager screenshot
- `storage.png` - Storage cleaner screenshot

**Recommended:**
- Resolution: 1920x1080 or higher
- Format: PNG or JPG
- Show the app in action with real data

You can take screenshots using:
- Windows Snipping Tool (Win + Shift + S)
- Or any screenshot tool of your choice

After adding screenshots, they will automatically appear in the README.md file on GitHub!
